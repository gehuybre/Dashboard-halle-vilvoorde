# Dashboard Halle-Vilvoorde

Een interactief dashboard met data over de regio Halle-Vilvoorde.

ðŸ”— [Bekijk het dashboard](https://share.streamlit.io) (link wordt toegevoegd na deployment)

## Lokaal uitvoeren

```bash
# Clone de repository
git clone https://github.com/gehuybre/Dashboard-halle-vilvoorde.git
cd Dashboard-halle-vilvoorde

# Installeer dependencies
pip install -r requirements.txt

# Start het dashboard
streamlit run dashboard.py
```

## Data

Het dashboard bevat informatie over:
- Gebouwenpark
- Huishoudens
- Bouwvergunningen
- Vastgoedprijzen

## Technologie

- Python 3.10+
- Streamlit
- Pandas
- Plotly
